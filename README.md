# streamlit-based-AI-Assistant-for-Data-Science
Implemented a natural-language query chatbot capable of understanding questions like summary, statistics, plotting, unique counts, and column-specific analysis using regex-based parsing,Added automatic dataset profiling, including missing value analysis, type inference, variance ranking, and categorical distribution summaries.
